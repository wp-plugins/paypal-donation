== Description ==
This PayPal Donation WordPress Plugin gives high level of flexible to admin to share some of the real information for donation. 

Through this plugin admin can,

* Able to add customize title for widget
* Can add up to three image to display in widget based on the percentage of donation received.
* Easy to add up to five different types of expenses.
* PayPal code box to enter the PayPal donation code.
* User can able to select around four type of Currency. [Kindly request if needed more...]

For more information : http://buffercode.com/project/paypal-donation/

== Installation ==

1. Upload the "paypal-donation" directory to the plugins directory.
2. Go to the plugins setting page and activate "paypal donation"
3. Go to Appearance --> Widget --> Drag paypal donation Widget to appropriate location
4. Go to Setting Page then Add the expenses, upload the images, amount received information.
5. Do save.

For more information : http://buffercode.com/project/paypal-donation/
